# Dashboard Interativo – Power BI

O dashboard interativo foi desenvolvido no Power BI com o objetivo de consolidar
e visualizar os principais indicadores financeiros analisados no estudo.

## Funcionalidades

- Visualização comparativa dos indicadores ROA, ROE e Margem Líquida;
- Análise por exercício financeiro (2023 e 2024);
- Gráficos de dispersão para avaliação das relações entre os indicadores;
- Filtros interativos por empresa, ano e indicador.

## Fonte dos dados

Os dados utilizados no dashboard foram previamente processados e tratados por meio
de scripts em Python, conforme descrito na metodologia do TCC, e importados no Power BI
em formato CSV.

## Observações

O arquivo do dashboard (`.pbix`) encontra-se disponível neste repositório exclusivamente
para fins acadêmicos e de reprodutibilidade do estudo.
